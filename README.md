## Missing `touying`?

Don't forget to run `git submodule init; git submodule update` to get access to the latest version of `touying`!

## Installing fonts

To install the Arial font in Ubuntu, run

```
sudo apt install ttf-mscorefonts-installer
sudo fc-cache -vr
```
